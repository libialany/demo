### **Desafío 2: Sección de Biografía**

**Tarea**: Crear la primera sección de la página donde irá la imagen y el texto.

1. Dentro del `div` contenedor, crea un nuevo `div` con la clase `bio`.
2. Dentro del `div` `bio`, agrega una imagen (`<img>`) con el atributo `src` apuntando a esta URL: `https://placekitten.com/200/200`.
3. Agrega un encabezado (`<h3>`) en el texto debe ir tu nombre.
4. Debajo del encabezado h3, agrega otro encabezado (`<h6>`) con algún texto, como "Desarrollador Web | Musico".

```html
<div class="bio">
    <!-- Agregar una imagen con el atributo src para la foto de perfil -->
    <img src="https://placekitten.com/200/200" alt="Foto de perfil" class="profile-img">

    <!-- Agregar encabezados para el nombre y el título -->
    <h3>John Doe</h3>
    <h6>Desarrollador Web | Creador | Entusiasta de la Tecnología</h6>
</div>
```
